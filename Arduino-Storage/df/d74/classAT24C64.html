Detailed Description

The AT24C64 provides 65,536 bits of serial electrically erasable and programmable read only memory (EEPROM) organized as 8192 words of 8 bits each. 32-Byte page write mode.

See Atmel Product description (Rev. 0336K-SEEPR-7/03), www.atmel.com/images/doc0336.pdf
