Detailed Description

template<uint16_t SIZE>
class Storage::Stream< SIZE >

Stream of given size on given storage. Write/print intermediate data to the stream that may later be read and transfered. Multiple stream may be created on the same device by assigning start address and size.

Parameters
[in]SIZEnumber of bytes in stream.
