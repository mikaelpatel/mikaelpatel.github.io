Detailed Description

template<BOARD::pin_t PIN>
class DS18B20< PIN >

Driver for the DS18B20 Programmable Resolution 1-Write Digital Thermometer.

Circuit

DS18B20
(VCC)--[4K7]--+ +------------+
(GND)---------)-----1-|GND |\
(Dn)----------+-----2-|DQ | |
(VCC)---------------3-|VDD |/
+------------+
