Detailed Description

template<BOARD::pin_t TX_PIN>
class ASO< TX_PIN >

Asynchronous Serial Output (ASO) template class using GPIO. Acts as an output only Stream, i.e. SoftwareSerial. May be used with baudrates up to 250.000 bps.

Parameters
[in]TX_PINboard pin for transmit signal.
