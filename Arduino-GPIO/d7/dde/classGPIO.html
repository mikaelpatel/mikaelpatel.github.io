Detailed Description

template<BOARD::pin_t PIN>
class GPIO< PIN >

General Purpose Digital I/O pin template class. Highly optimized pin access. The PIN address is 12-bit control register address (msb) and 4-bit bit number (lsb). See Board.h for definitions.

Parameters
[in]PINboard pin definition.
