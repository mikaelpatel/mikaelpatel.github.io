Detailed Description

template<BOARD::pin_t SS_PIN, uint32_t FREQ = 16000000L>
template<uint16_t SIZE>
class SRAM< SS_PIN, FREQ >::Stream< SIZE >

SRAM based Stream of given size. Write/print intermediate data to the stream that may later be read and transfered. Multiple stream may be created on the same device by assigning start address and size.

Parameters
[in]SIZEnumber of bytes in stream.
