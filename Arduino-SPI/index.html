Arduino-SPI

The Serial Peripheral Interface (SPI) library is an abstract interface for bus managers and device driver support (SPI::Device).

The library contains bus manager implementations for hardware resources (Hardware::SPI) and using Ardino-GPIO (Software::SPI).
