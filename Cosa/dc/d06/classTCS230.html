Detailed Description

Cosa Device Driver for TCS230 Programmable Color Light-to-Frequency Converter.

Circuit

TCS230 Module
P1 +------------+ P2
(D4)---------------1-|S0 S3|-1-----------------(D7)
(D5)---------------2-|S1 S2|-2-----------------(D6)
3-|OE OUT|-3------------(D3/EXT1)
(GND)--------------4-|GND VCC|-4----------------(VCC)
+------------+
