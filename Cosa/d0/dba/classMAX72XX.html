Detailed Description

MAX72XX Serially Interfaced, 8-Digit LED Display Drivers, device driver for IOStream access.

The MAX7219/MAX7221 are compact, serial input/output common-cathode display drivers that interface microprocessors (Î¼Ps) to 7-segment numeric LED displays of up to 8 digits, bar-graph displays, or 64 individual LEDs. Included on-chip are a BCD code-B decoder, multiplex scan circuitry, segment and digit drivers, and an 8x8 static RAM that stores each digit.
