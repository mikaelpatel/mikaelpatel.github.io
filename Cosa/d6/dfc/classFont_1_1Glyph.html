Detailed Description

Display a character. Used in the form: Font::Glyph glyph(font, character); for (uint16_t i = 0; i < HEIGHT; i += 8) { for (uint16_t j = 0; j < WIDTH; j++) { uint8_t bits = glyph.next(); // display bits } }
