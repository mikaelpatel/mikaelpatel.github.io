Detailed Description

IO handler for HD44780 (LCD-II) Dot Matix Liquid Crystal Display Controller/Driver when using the MJKDZ IO expander board based on PCF8574 I2C IO expander device driver.

Circuit

PCF8574/MJKDZ
+-----U------+
(GND)---[ ]---------1-|A0 VCC|-16--------------(VCC)
(GND)---[ ]---------2-|A1 SDA|-15-----------(SDA/A4)
(GND)---[ ]---------3-|A2 SCL|-14-----------(SCL/A5)
(LCD D4)------------4-|P0 INT|-13
(LCD D5)------------5-|P1 P7|-12-----------(LCD BT)
(LCD D6)------------6-|P2 P6|-11-----------(LCD RS)
(LCD D7)------------7-|P3 P5|-10-----------(LCD RW)
(GND)---------------8-|GND P4|-9------------(LCD EN)
+------------+
